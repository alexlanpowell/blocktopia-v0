# Music Track Assets

Place your background music MP3 files here. Required files:

1. `ambient.mp3` - Default calm background track (2-3 min loop)
2. `default-saloon.mp3` - Old-timey honky-tonk saloon music cosmetic pack (2-3 min loop)
3. `electronic.mp3` - Upbeat EDM cosmetic pack (2-3 min loop)

**Recommended Sources:**
- Incompetech.com (Kevin MacLeod - attribution required)
- Purple Planet Music (Free with attribution)
- Bensound (Free with attribution or license purchase)

**File Specifications:**
- Format: MP3
- Sample Rate: 44.1kHz
- Bitrate: 192kbps
- Channels: Stereo
- Size: 2-4MB each
- Loop: Seamless (no gaps at start/end)

**Note:** The app will work without these files (graceful degradation), but you should add them for the full experience.
