# Sound Effects Assets

This directory contains the game's sound effects. The AudioManager will load available sounds and gracefully skip missing ones.

2. `piece_place.mp3` ✓ - Block placement thud (67 KB)
3. `line_clear.mp3` ✓ - Single row clear chime (74 KB)
4. `game_over.mp3` ✓ - Game over sound (26 KB)
