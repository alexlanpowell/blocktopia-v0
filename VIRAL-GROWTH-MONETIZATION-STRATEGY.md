**Formula for Viral TikTok Ads:**
1. **First 3 seconds = EVERYTHING**
   - Hook: "I can't stop playing this game..."

   - Challenge: "Bet you can't beat my score"
   
2. **Middle 10 seconds = Proof**

## Ad Placement Blueprint {#ad-placement-blueprint}

### Complete Ad System Architecture

```
HOME SCREEN:
‚îú‚îÄ Banner Ad (bottom, always visible)
‚îî‚îÄ Daily Bonus Chest (watch ad for 3x reward)

GAME SCREEN:
‚îú‚îÄ Banner Ad (bottom, always visible)
‚îî‚îÄ Mid-Game Power-Up Offer (when board 70% full)

GAME OVER SCREEN:
‚îú‚îÄ Continue (watch ad to keep playing) [Existing]
‚îú‚îÄ Double Score (watch ad for 2x score) [NEW]
‚îú‚îÄ Unlock Piece (watch ad every 5th game) [NEW]
‚îî‚îÄ Interstitial Ad (after "Play Again" every 3 games) [Existing]

MENUS:
‚îî‚îÄ No ads (keep clean UX)
```

### Ad Frequency Rules

**Global Rules:**
- Max 1 ad per 3 minutes (prevents spam)
- Never show ad during active gameplay
- Always show clear reward before ad
- Can decline any ad (except interstitial)
- Track ad fatigue (reduce frequency if user declining often)

**Per-Session Limits:**
```typescript
interface AdLimits {
  maxAdsPerSession: 10;
  minTimeBetweenAds: 180; // 3 minutes in seconds
  maxInterstitialsPerDay: 5;
  maxRewardedPerDay: 20;
}
```

**Priority System** (if multiple ads ready):
1. Rewarded ads (user-initiated) - ALWAYS show
2. Daily chest - Show once per day
3. Double score - Show if game just ended
4. Power-up booster - Show if board critical
5. Interstitial - Show only if frequency rules met

---

### User Experience Optimization

**Ad Quality Standards:**
- Load ads in background (pre-cache)
- Never freeze app waiting for ad
- If ad fails to load: Give reward anyway (goodwill)
- Show loading indicator if ad takes >2 seconds
- Allow "close" button after 5 seconds (rewarded) or 30 seconds (interstitial)

**Testing Checklist:**
- [ ] Ads load within 2 seconds
- [ ] Reward given immediately after ad
- [ ] No crashes when ad closes
- [ ] Works on slow connections
- [ ] Works in all supported countries
- [ ] Respects premium user status

## Case Study: What NOT to Do

### Common Mistakes (Learn from others' failures)

**1. Too Many Forced Ads**
- Showing interstitial every game = uninstall
- Users tolerate 1 forced ad per 5-10 minutes max
- Survivor.io has ZERO forced ads (all rewarded)

**2. Poor Ad Timing**
- Mid-game interstitials = rage quit
- Ads during loading = frustration
- Only show ads at natural breaks

**3. Ignoring Reviews**
- Negative reviews compound
- Respond to every review (especially negative)
- Fix issues fast

**4. Spending Too Much Too Soon**
- Don't spend $10K on ads in Month 1
- Test small, learn, then scale
- Most indie devs go broke from over-spending on ads

**5. No Metrics Tracking**
- Flying blind = wasted money
- Set up analytics BEFORE launching ads
- Make data-driven decisions

**6. Expecting Instant Success**
- Takes 6-12 months to see real traction
- Keep iterating, don't give up
- Most successful apps took years to blow up
