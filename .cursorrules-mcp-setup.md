# MCP Supabase Configuration for Blocktopia

## What is MCP?
Model Context Protocol (MCP) allows AI assistants (like me!) to directly query your Supabase database for debugging and data inspection.

## Setup Instructions

### Option 1: Global MCP Configuration (Recommended)

Your MCP config file location:
```
Windows: %APPDATA%\Cursor\User\globalStorage\mcp.json
Mac: ~/Library/Application Support/Cursor/User/globalStorage/mcp.json
```

### Step 1: Get Your Database Connection String

1. Go to your Supabase dashboard: https://supabase.com/dashboard
2. Select "Blocktopia" project
3. Go to **Settings** → **Database**
4. Scroll to **"Connection string"** section
5. Click **"URI"** tab
6. Copy the connection string (looks like):
   ```
   postgresql://postgres:[YOUR-PASSWORD]@db.ue1cwmpkoexmlgqxkdt.supabase.co:5432/postgres
   ```
7. Replace `[YOUR-PASSWORD]` with your actual database password

### Step 2: Create/Edit MCP Config

Add this to your `mcp.json` file:

```json
{
  "mcpServers": {
    "supabase-blocktopia": {
      "command": "npx",
      "args": [
        "-y",
        "@modelcontextprotocol/server-supabase",
        "postgresql://postgres:YOUR_DB_PASSWORD@db.ue1cwmpkoexmlgqxkdt.supabase.co:5432/postgres"
      ]
    }
  }
}
```

**Important:** Replace `YOUR_DB_PASSWORD` with your actual database password from Supabase!

### Step 3: Restart Cursor

After saving the MCP config, fully restart Cursor for changes to take effect.

## How to Find Your Database Password

Your database password is the one you set when creating the Supabase project.

**If you forgot it:**
1. Go to Supabase Dashboard → Settings → Database
2. Click "Reset Database Password"
3. Set a new password
4. Update the MCP config with the new password
